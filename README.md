# 🎬 Movie Search Web App

## 📖 Description
A responsive movie search web application developed using HTML, CSS and JavaScript with API integration for real-time movie search and movie information display.

---

## ✨ Features
- 🎥 Real-time movie search
- 🔗 API integration
- 📱 Responsive design
- ⚡ Dynamic movie details
- 🎨 Modern UI interface

---

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Movie API
